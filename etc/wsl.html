wsl document
…

    • 윈도우 cmd 또는 powershell 실행 후

    • wsl -install (설치 후 재시작)

  2. 설치 확인 및 실행

    • 윈도우 리부팅 후 cmd 창에서 아래 명령어 실행

    • wsl -l -v

  3. 윈도우 터미널 설치

    • MS store에서 윈도우 터미널 다운로드 및 설치

    • Remote-SSH
