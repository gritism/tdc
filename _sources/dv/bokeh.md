# Bokeh

* Data Visualization Tool, poltly가 가장 많이 사용됨
* jupyter notebook, web browser 등에서 visualization 가능

1. 설치
    - `pdm add bokeh`

2. 실행 (bokeh server)
    - bokeh server는 웹 서버로 생성된 차트를 브라우저 상에서 조회
    - `pdm run bokeh --dev --show stock.py`
      ```{note}
        > --dev: 개발자 모드 실행, 파일 수정 시 auto reload
      ```
